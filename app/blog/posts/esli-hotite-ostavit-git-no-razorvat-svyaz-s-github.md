  body: >-
    Этот способ для тех, кто хочет оставить локальный контроль версий (коммиты),
    но убрать привязку к конкретному удаленному репозиторию на GitHub.


    1.  Откройте **терминал** в VS Code.
        
    2.  Выполните команду, которая удалит ссылку на удаленный репозиторий
    (origin):
        
        ```
        git remote remove origin
        ```

        
    3.  После этого ваш проект останется Git-репозиторием, но VS Code перестанет
    предлагать вам синхронизироваться с GitHub (пушить, пуллить), так как "точка
    входа" будет удалена
